# Erabiltzaileak bete beharreko baldintzak 

Windows ->
1. Windows SE-an lan egingo badu WSL (Windows Subsystem for Linux) izan beharko du. https://docs.microsoft.com/es-es/windows/wsl/install-win10
2. Ubuntu aplikazioa (Microsoft Store-n lortu daiteke) instalatuta eduki beharko du.
3. WhatWeb instalatuta izan. Lehen aipatutako errepositoriotik eskuratu daiteke. 
4. Gure paketea jaitsi eta .exe exekutatu

Linux Mint/Ubuntu/Debian/Kali Linux ->
1. Terminal bat ireki
```bash
sudo apt install whatweb
```
Gure paketea jaitsi eta .deb exekutatu

MacOS ->
1. whatweb instalatu https://github.com/urbanadventurer/WhatWeb/wiki/Installation#installing-in-macos-catalina
